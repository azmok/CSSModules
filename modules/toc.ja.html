  1. Named Charactor Reference
  2. Hexadecimal Numeric Charactor Reference
  3. Decimal Numeric Charactor Reference

Unicode in JS

There are three ways to input charactors indirectly as string in source code

  1. Ascii Hexadecimal Escape Sequence
  2. UTF16 Hexadecimal Escape Sequence
  3. Unicode Code Point Escape Sequence

**

1. Ascii Hexadecimal Escape Sequence

\x 2HexDigits

2HexDigits is two hexadecimal digits, e.g.

`\x3c' === '<`

Two-Hex-digits is corresponging hex digits and chars in ascii table **
